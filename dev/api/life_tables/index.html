Life Tables

Functions for converting between life table quantities and extracting life table columns from matrix projection models.

Conversions Between Life Table Columns

MatrixProjectionModels.lx_to_px — Function
lx_to_px(lx::AbstractVector)

Convert survivorship (lx) to survival probability (px). px[x] = lx[x+1] / lx[x]. Last value is 0 (absorbing).

source
MatrixProjectionModels.lx_to_hx — Function
lx_to_hx(lx::AbstractVector)

Convert survivorship (lx) to hazard rate (hx). hx[x] = -log(px[x]). Last value is Inf (absorbing).

source
MatrixProjectionModels.px_to_lx — Function
px_to_lx(px::AbstractVector)

Convert survival probability (px) to survivorship (lx). lx[1] = 1, lx[x+1] = lx[x] * px[x].

source
MatrixProjectionModels.px_to_hx — Function
px_to_hx(px::AbstractVector)

Convert survival probability (px) to hazard rate (hx). hx[x] = -log(px[x]).

source
MatrixProjectionModels.hx_to_lx — Function
hx_to_lx(hx::AbstractVector)

Convert hazard rate (hx) to survivorship (lx). lx[1] = 1, lx[x+1] = lx[x] * exp(-hx[x]).

source
MatrixProjectionModels.hx_to_px — Function
hx_to_px(hx::AbstractVector)

Extraction from MPMs

MatrixProjectionModels.mpm_to_lx — Function
mpm_to_lx(matU::AbstractMatrix; start::Int=1, xmax::Int=1000, lx_crit::Real=0.01)

Calculate age-specific survivorship (lx) from a survival matrix U. lx[x] is the probability of surviving from the start stage to age x.

Arguments

  • matU: Survival/growth transition matrix (n × n)
  • start: Index of the starting stage (default: 1)
  • xmax: Maximum age to compute (default: 1000)
  • lx_crit: Stop when lx drops below this value (default: 0.01)

Returns

Vector of survivorship values, starting with lx[1] = 1.0.
