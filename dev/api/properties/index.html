Matrix Properties

Functions for testing structural properties of projection matrices.

StructuredPopulationCore.is_ergodic — Function
is_ergodic(A::AbstractMatrix)

Test if matrix A is ergodic (irreducible and aperiodic/primitive). An ergodic matrix converges to a unique stable distribution.

source
StructuredPopulationCore.is_irreducible — Function
is_irreducible(A::AbstractMatrix)

Test if matrix A is irreducible. A non-negative matrix is irreducible if its associated directed graph is strongly connected. Equivalent: (I + |A|)^(n-1) has all positive entries.

source
StructuredPopulationCore.is_primitive — Function
is_primitive(A::AbstractMatrix)

Test if non-negative matrix A is primitive. A non-negative irreducible matrix is primitive iff it is aperiodic. Equivalent: A^(n^2 - 2n + 2) has all positive entries (for irreducible A).

source
MatrixProjectionModels.is_leslie — Function
is_leslie(A::AbstractMatrix)

Test if matrix A has Leslie structure:

  • Non-negative entries in first row (fecundity)
  • Positive entries on sub-diagonal (survival)
  • All other entries are zero
